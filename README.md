# ShadowFox Data Science Internship
## Intermediate Level: Delhi AQI Analysis

### Overview
This project analyzes the Air Quality Index (AQI) of Delhi, focusing on pollutant dynamics and seasonal variations.

### Workflow
1. **Data Cleaning**: Handled missing values using median imputation.
2. **Statistical Analysis**: Descriptive profiling of pollutants like PM2.5 and NO2.
3. **Visualizations**: Created trend lines and heatmaps to show pollution hotspots.
